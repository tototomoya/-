
勤怠管理アプリ
	
	概要

		rubyプログラムで作った勤怠データから、JavaScriptでグラフィカルな勤怠表を作成するアプリです。

	ユースケース
		
		スタッフが出した希望シフト(一時間単位）から勤怠表を作成します。
			
		スタッフには一人一人にIDを振り、新規スタッフにはblankIDから新しく振り当てています。
			
	工夫した点
	
		勤務時間のデータ表現を工夫しました。
