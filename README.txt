勤怠管理アプリ
	
	機能概要

		バックエンドでCGIとしてrubyプログラムを動かし、結果を保存したJSONファイルからSVGでグラフィカルな表を作成するアプリです。

	ユースケース
		
		スタッフが出した希望シフト(一時間単位）から勤怠表を作成します。
			
			同じ時間帯に入れるスタッフの数は二人までです。
			
			スタッフには一人一人にIDを振り、新規スタッフにはblankIDから新しく振り当てます。
			
	工夫した点
	
		SVGでグラフが作りやすくなるように、勤務時間をデータ表現する方法を工夫しました。
		
	製作期間
	
		設計から実装まで三日ほどです。(一日4～5時間）
